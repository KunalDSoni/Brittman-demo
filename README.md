# Brittman — Complete Corporate Services

Offline, self-hosted version of the Brittman corporate-services website, built on a
Framer template with all content sourced from [brittman.com](http://www.brittman.com/).
Theme, images, colours and typography are unchanged from the template — only the text
content was replaced with Brittman's.

## Notes

The Framer template renders its content twice: a static HTML snapshot and a client-side
React re-render driven by JavaScript module bundles (`*.mjs`) and binary CMS data
(`*.framercms`). Brittman content was applied to **all three layers** so the page never
flashes the original template text before settling.

Served from the repository root for GitHub Pages (`.nojekyll`).
